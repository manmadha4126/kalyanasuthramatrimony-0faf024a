# Kalyanasuthra Matrimony

> South India's most trusted matrimony service for Telugu, Tamil & Kannada families.

## About

Kalyanasuthra Matrimony is a premium matchmaking platform offering dedicated relationship managers, 15,000+ verified profiles, and personalized matchmaking services based in Tirupati, Andhra Pradesh.

**Website**: [kalyanasuthramatrimony.com](https://kalyanasuthramatrimony.com)

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite 5
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend**: Cloud-hosted (PostgreSQL, Auth, Edge Functions, Storage)
- **AI**: Integrated AI chatbot for customer assistance

## Local Development

Requires Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project
cd kalyanasuthra-matrimony

# Install dependencies
npm i

# Start development server
npm run dev
```
